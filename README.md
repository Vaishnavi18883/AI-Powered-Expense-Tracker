
# AI-Powered Expense Tracker

AI-powered expense and income tracking web application built with React.js.  
The application provides smart saving tips and budget warnings using Google Gemini API, along with interactive financial insights through charts.

## Features

- Track income and expenses
- Category-wise budgeting
- Smart budget alerts
- AI-generated saving tips
- Interactive charts for financial visualization

## Tech Stack

- React.js
- Tailwind CSS
- Chart.js
